# Ember-ui-components

This ember-cli addon is a collection of common user interface components.

## Demo

http://lozjackson.github.io/ember-ui-components/

## Installation

* `ember install ember-ui-components`

## Components

* Button
* ButtonGroup
* CheckBox
* ContextMenu
* Dropdown
* FormElement
* Fieldset
* Input
* ModalDialog
* ModalWindow
* Select
* SlideMenu
* Switch
* Tabs


## Services

* Dialog

## CSS Classes

* Button
* ButtonGroup
* FormElement

* Panel
